> 版本: v1.0 | 状态: Final | 最后更新: 2026-05-10

---

## 1. 产品概述

### 1.1 产品定位

**一句话定义：** 让中小企业用「上传文件 → AI诊断 → 一键清洗 → 下载结果」四步，解决所有表格数据脏乱差问题。

**差异化价值主张：**

| 对比维度 | 传统方案 | 我们的方案 |
|---------|---------|-----------|
| 人力方案 | 雇人手动整理Excel，慢、贵、易出错 | 上传即诊断，分钟级完成 |
| 专业工具 | ETL工具需技术人员操作，学习成本高 | AI理解数据语义，非技术人员也能上手 |
| 大公司服务 | 收费高、门槛高，不愿接小单 | 按数据量计费，中小企业用得起 |

**核心差异化：** AI不仅仅做格式校验，而是理解数据语义——知道"2024/1/5"是一个日期而非文本，识别"138-1234-5678"是手机号而非普通数字。

### 1.2 目标市场

- **目标客户：** 年营收500万-5000万的中小企业，IT团队不足3人
- **典型行业：** 零售电商、制造业、批发贸易、物流仓储
- **核心场景：** 财务对账、订单整合、库存迁移、客户数据清洗

### 1.3 产品愿景

让数据清洗像洗照片一样简单——上传、诊断、一键处理。成为中小企业数据治理的入口产品。

### 3.2 智能诊断模块

**前置校验：**
- 拖拽上传（支持 .xlsx, .xls, .csv, .tsv）
- 文件类型白名单校验（扩展名 + 魔数检测）
- 编码自动检测（UTF-8 / GBK / Shift-JIS）
- 多Sheet选择（仅第一个Sheet为默认）

**诊断功能：**
- 自动识别表头行位置
- 逐列扫描生成数据profile（每列）：
  - 数据类型推断（日期/数字/金额/手机号/身份证/邮箱/URL/地址/文本）
  - 空值率 & 缺失分布
  - 完全重复行检测
  - 异常值检测（数值列用IQR，文本列用长度异常）
  - 格式一致性（日期/手机号/金额格式占比分析）
  - 基数统计（唯一值个数）
- 整体数据健康评分（0-100）
- AI增强：基于采样数据的语义理解建议

**输出：** 数据健康报告（每个列一个卡片，按严重程度排序）

### 3.3 清洗方案模块

**AI方案生成：**
- 基于diagnosis profile + 采样数据 → LLM理解列语义 → 生成清洗建议
- 建议内容：问题描述、建议动作、影响行数、置信度

**支持的清洗动作：**

| 动作 | 说明 | 参数 |
|------|------|------|
| remove_duplicates | 按指定列去重 | 保留策略（首条/最新） |
| fill_null | 填充空值 | 填充值/策略（均值/众数/固定值） |
| format_date | 统一日期格式 | 目标格式（默认 yyyy-MM-dd） |
| format_number | 清除文字转纯数字 | 替换模式 |
| trim_whitespace | 去除空格 | 全部/前后 |
| clean_phone | 手机号格式化 | 去除分隔符 |
| split_column | 拆分列 | 分隔符、目标列名 |
| merge_columns | 合并列 | 分隔符 |
| remove_outliers | 移除异常值 | IQR倍数 |

**用户交互：**
- 每项操作可单独勾选/取消
- 参数可自定义修改
- 支持另存为"清洗模板"

### 3.4 执行与导出模块

**执行：**
- 按序执行已确认的清洗动作
- 实时进度推送（WebSocket）
- 操作日志记录（每步的 before/after 统计）

**结果预览：**
- 清洗前后对比视图（左右分栏）
- 差异行高亮
- 关键指标变化（行数、完整度、健康分）

**导出：**
- .xlsx 格式（保留多Sheet）
- .csv 格式（UTF-8 BOM，兼容Excel中文）
- 清洗报告PDF（操作记录 + 前后对比审计）

### 3.5 账户与计费模块

**认证：**
- 手机号 + 验证码登录（MVP: 任意6位数字即可登录）
- 无密码设计（降低使用门槛）

**套餐：**

| 套餐 | 价格 | 月处理行数 | 私有部署 | 适用场景 |
|------|------|-----------|---------|---------|
| 免费体验 | ¥0 | 1,000行 | 否 | 尝鲜试用 |
| 基础版 | ¥99/月 | 5万行 | 否 | 偶尔使用 |
| 专业版 | ¥299/月 | 50万行 | 否 | 日常运营 |
| 企业版 | ¥999/月 | 不限行数 | 可选 | 高频使用 |

**计费规则：**
- 按处理行数计费（去重前的原始行数）
- 当月未用完额度不结转
- 超额后可升级套餐或等待下月重置

**数据生命周期：**
- 上传后在服务器保留7天，自动删除
- 用户可手动提前删除
- 删除操作不可逆

## 4. 用户核心流程

```
[A] 新用户首次使用

  1. 访问首页 → 点击"开始使用"
  2. 手机号验证码登录（无需注册，自动创建账号）
  3. 进入工作台 → 点击上传区域
  4. 上传文件（或拖拽）
  5. 等待自动诊断（5-30秒，根据文件大小）
  6. 查看数据健康报告
  7. 点击"下一步生成方案"
  8. 查看AI建议，逐项确认/调整
  9. 点击"执行清洗"
  10. 实时进度 → 预览清洗结果
  11. 下载结果文件 + 清洗报告PDF
  12. 完成任务

[B] 老用户复用模板

  1. 登录 → 进入"我的模板"
  2. 选择历史模板 → 上传新文件
  3. 模板自动匹配列名 → 微调确认
  4. 执行 → 下载 → 完成
```

## 5. MVP 范围

### 5.1 包含（V1）

- 单文件上传（Excel/CSV/TSV，最大100MB）
- 规则诊断引擎（缺失/重复/格式/异常值）
- AI辅助诊断（采样+LLM分析，增强建议）
- 9种核心清洗动作（去重、填充、日期格式统一、金额去单位、去空格、手机号清洗、列拆分、列合并、异常值移除）
- 清洗方案确认页（勾选+参数调整）
- 执行 + WebSocket进度显示
- 清洗前后对比预览
- 导出 .xlsx / .csv / 清洗报告PDF
- 手机号验证码登录（自动注册）
- 套餐计费（免费版 / 基础版 / 专业版 / 企业版）
- 数据7天自动过期清理
- 清洗模板保存和复用

### 5.2 不包含（V2+）

- 多文件关联清洗/合并
- 多Sheet同时处理
- 实时协同/分享
- 开放API
- 私有化部署支持
- 对接飞书/钉钉多维表格
- 模糊匹配/近似去重（仅做精确去重）
- 数据可视化分析仪表盘
